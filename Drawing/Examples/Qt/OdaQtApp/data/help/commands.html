Command Reference

This Command Reference includes information about each command that is available in $(TD_APP_NAME) from the menus or from the command line (command line text is in italics).

File Commands

New (new): Creates a new file (supported for .dwg files only). Use startup to change behaviour of it.

Open (open): Opens a .dwg, .dgn, or .dwf file.

Close (close): Closes the current file.

Recover (recover): Opens and attempts to repair a .dwg file. (Command line only.)

Audit (audit): Checks the open .dwg file for errors and reports results with an option to repair any errors that are found. (Command line only.)

Save (qsave): Saves the open .dwg or .dgn file in its same file format.

Save As (saveas): Saves the open .dwg or .dgn file with a new name, in a new location, or in a new file format.

Import DGN/DWF (import): Imports the file contents into the currently opened .dwg file. (Command line only.)

Insert Raster Image (insertraster): Inserts a raster image to a .dwg file. (Command line only.)

Plot (plot): Prints the open .dwg file according to the options you select (supported for .dwg files only).

Exit (quit): Closes $(TD_APP_NAME).

Edit Commands (command line only)

Undo (undo): Reverses the last operation performed in a .dwg file (supported for .dwg files only). You can also press CTRL + Z.

Redo (redo): Reverses the undo command (supported for .dwg files only). You can also press CTRL + Y.

Select (select): Selects one or more items in a .dwg or .dgn file. You can click an item or its corner to select it.

Deselect (ai_deselect): Removes selection from all items in a .dwg or .dgn file. You can also press ESC to remove selection.

Erase (erase): Removes an item you select from a .dwg or .dgn file.

Isolate Objects (isolateobjects): Displays the selected objects and hides other objects.

Array Editing (arrayedit): Edits array of objects and their source objects. Number and types of operations depend on array type (for example, number of rows, columns, levels, distance between them, tangent changes, etc.).

Text Converting (txtexp): Converts text to lines or triangles using vectorization.

View Commands

Model or Layout (qvlayout): Displays the designated tab when you click it. (Command line and mouse only.)

Regen (regenall): Regenerates the current view of the drawing.

Zoom Window (zoom or zoom w): Displays a window view of the drawing by specifying two opposite corners.

Zoom In (zoom or zoom 2x): Enlarges the view of the drawing. You can also roll the mouse wheel button to zoom in and out of the drawing.

Zoom Out (zoom or zoom .5x): Reduces the view of the drawing. You can also roll the mouse wheel button to zoom in and out of the drawing.

Zoom Extents (zoom or zoom e): Displays the drawing at its extents. You can also double-click the mouse wheel button to zoom to the drawing extents.

Pan (pan): Moves the view around the drawing by clicking and dragging the mouse. You can also click the mouse wheel button to pan the drawing without choosing the command.

Constrained Orbit (3dorbit): Rotates the view around a three dimensional axis. Click and hold the mouse button to rotate the view. You can also press SHIFT + the mouse wheel button to start 3d orbit. (Available only in model space.)

Free Orbit (3dforbit): Rotates the view in three dimensions without axis restriction. You can also press CTRL + SHIFT + the mouse wheel button to start 3dforbit. (Available only in model space.)

Continuous Orbit (3dcorbit): Rotates the three dimensional view continuously. Click and hold the mouse button to rotate the view, then release the mouse button to continue the rotation. To stop the rotation, click in the drawing or press Esc or Enter. (Available only in model space.)

3D Views (view): Displays the drawing from the named view. You can also delete and change the properties of saved views.

Visual Styles (vscurrent): Changes the visual style of current view.

3D Background Color (background): Modifies the color of the background that displays in model space.

Format Commands

Properties (properties): Displays the Properties palette where you can modify general properties of selected entities. All properties of line and arcs can be modified, but only general properties and material properties can be modified for other entities. Enter propertiesclose to close the Properties palette. (Command line only.)

Layer (layer): Displays and modifies the layer properties of the drawing.

Tools Commands

Command Line (commandline or commandlinehide): Displays the command line window if it has been closed. You can also press CTRL + 9 to show or hide the command line window.

Set System Variable (setvar): Allows you to view or set the value for any system variable defined for the current .dwg or .dgn file. (Command line only.)

Reset System Registry Variables (resetvars): Resets system variables to their default values. (Command line only.)

Registry Variable Discard Back Faces (discardbackfaces): When turned on (set to 1), surface geometry is visible from one side only. This is turned off (set to 0) by default. (Command line only.)

Registry Variable Custom Toolbar Path (ddcuipaths): Specifies a folder from which to load toolbars defined in custom.cui files. For example, the toolbar defined in OdaQtObjects.cui is loaded automatically from resources, but the .cui file can be copied, customized, and its location specified using DDCUIPATHS. (Command line only.)

Load Menu (menu or cuiload or cuiunload): Menu is loaded from .cui file, which can be customized. (Command line only.)

Load Application (appload): Loads a TX application (.tx file) and runs it within $(TD_APP_NAME). You can also create a Startup Suite, which allows you to define what applications load automatically when starting $(TD_APP_NAME). Windows users and ODA members can load custom .NET TX plug-ins.

Registered Commands: Runs the command selected from a list of currently loaded commands.

Run Script (script): Runs a selected script (.scr file).

Draw Commands

Modeling Box (box): Draws a box.

Modeling Sphere (sphere): Draws a sphere.

Line (line): Draws a line.

Circle (circle): Draws a circle using the specified center and radius or diameter. (Command line only.)

Array (array): Creates an array of copies of selected objects. (Command line only.)

Available options:

  • Rectangular — Creates rows and columns from copies of selected objects.
  • Path — Places the copies of selected objects along a specified path.
  • Polar — Places the copies of selected objects around a specified point.

Text (text): Draws a single-line text object and specifies its start point, height and angle. (Command line only.)

Window Commands

Close (close): Closes the current window.

Close All (closeall): Closes all open windows.

Cascade (syswindows cascade): Arranges all open windows so they overlap.

Tile Horizontally (syswindows hor): Arranges all open windows horizontally.

Tile Vertically (syswindows vert): Arranges all open windows vertically.

Arrange Windows (syswindows arrange): Arranges all open windows so they are all visible.

Help Commands

Help (help): Displays documentation for $(TD_APP_NAME).

About (about): Displays information about $(TD_APP_NAME).
