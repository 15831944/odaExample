$(TD_APP_NAME) is a free cross-platform application provided by the ODA for the visualization of CAD drawing files. It incorporates the power and flexibility of ODA SDK into a single application that can render both .dwg files and .dgn files using a common command interface for both formats.

In addition to serving as a rendering tool, $(TD_APP_NAME) can be used as a test container application for TX developers. TX developers can load custom TX modules and run custom commands from the console, and can also customize the $(TD_APP_NAME) menus.

ODA members are utilizing the ODA SDK platform to create thousands of CAD and technical graphics applications across a wide variety of engineering domains. $(TD_APP_NAME) highlights only a small subset of the full platform capabilities.

Features

$(TD_APP_NAME) contains the following features:

  • Load and render .dwg, .dxf, .dgn and .dwf files.
  • Create, save, and print .dwg files.
  • Recover and audit damaged .dwg files.
  • Import .dgn files.
  • Work with architecture objects (uses ODA SDK Architecture integration, Windows only.)
  • Zoom, pan, orbit, and select various preset 3D views.
  • Select an active layout via drawing window tabs. Switch the current viewport.
  • Select and deselect items in a drawing. Delete items from a drawing. Use grips in .dwg files.
  • Undo and redo operations in .dwg files.
  • Draw boxes, spheres, and lines.
  • Edit general properties of entities; edit all properties of lines and arcs in .dwg files.
  • Edit layer properties using the Layer Properties Manager (turn layers on/off, etc.).
  • Use menu, toolbar, and command window interfaces.
  • Customize menus and toolbars via .cui files.
  • View and change system variables, and reset system variables.
  • Load ODA SDK Extension (TX) plug-ins on demand or explicitly via user request.
  • Load .NET TX plug-ins (Windows only, ODA members only).
  • Define in a Startup Suite the applications to load automatically when $(TD_APP_NAME) starts.
  • Enter arguments and switches when running the $(TD_APP_NAME) executable to automatically open files, run scripts, and load TX applications. Enter /? for a list of arguments, for example: "c:\$(TD_EXE_NAME).exe /?".
